# Style Snippets

This repository intentionally stores only short, attributed cues rather than full essays.
Use these snippets to calibrate openings, pivots, distinctions, analogies, caveats, and endings.
Do not splice them together into new prose.

## Openings

- `What to Do` (2025): `What should one do?` Start with a blunt, childlike big question.
- `Good Writing` (2025): `Let's examine it.` State the puzzle, then lean toward explanation.
- `Good Writing` (2025): `How can this possibly be true?` Follow a bold thesis with self-skeptical inquiry.
- `Founder Mode` (2024): `a question it raised` Use the anecdotal hook to uncover the real topic.
- `How to Do Great Work` (2023): `what would the intersection look like?` Ask for the shared structure across cases.
- `Earnestness` (2020): `Why does this matter so much?` Pivot from term-definition to stakes.
- `Schlep Blindness` (2012): `lying around unexploited` Signal hidden value in plain sight.
- `Writing, Briefly` (2005): `it generates them` Put the thesis in the opening, not the payoff.
